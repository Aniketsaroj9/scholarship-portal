# Requirements

## Scope Boundaries

**In Scope:**
- Applying a new, responsive modern UI design to all frontend web pages using Vanilla CSS or a framework.
- Converting all legacy inline db queries to Prepared Statements.
- Removing deprecated/legacy HTML code (like `<center>`, `bgcolor`, `<iframe>`).
- Improving directory and internal structure of PHP files.
- Modifying `setup.sql` if required for constraints, though table schemas usually stay similar.

**Out of Scope:**
- Creating a Dockerfile or deployment mechanisms.
- Changing the underlying database engine away from MySQL.
- Adding complex new business logic paths (e.g. email verification tokens that didn't exist before) unless functionally necessary for security.
